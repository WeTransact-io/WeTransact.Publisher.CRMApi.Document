Private Offers API

PrivateOfferController

6 endpoints

Manage private offers for AWS Marketplace products including creation, updates, deletion, and product/plan management.

GET /api/privateoffer Get all private offers

Description

Retrieves a paginated list of all private offers with optional filtering.

Query Parameters

Parameter Type Required Description
page integer Optional Page number for pagination
size integer Optional Number of items per page

Response Codes

200 Success - Returns list of private offers
500 Internal Server Error
GET /api/privateoffer/{id} Get private offer by ID

Description

Retrieves a specific private offer by its unique identifier.

Path Parameters

Parameter Type Required Description
id Guid Required Unique identifier of the private offer

Response Codes

200 Success - Returns private offer details
404 Private offer not found
500 Internal Server Error
POST /api/privateoffer Create new private offer

Description

Creates a new private offer with the provided details.

Request Body

Content-Type: application/json

{ "awsProductId": "string (required)", "privateOfferName": "string (required, max 150 chars)", "offerDescription": "string (max 255 chars)", "buyerAccountId": "string (required, 12 digits)", "contractStartingType": "enum (required)", "contractStartDate": "datetime (required)", "contractDuration": "number (required, 1-60)", "billingFrequency": "string (required)", "currency": "string (required)", "contractValue": "string (required)", "expirationDate": "datetime (required)", "usageEndDate": "datetime (required)", "pricingModel": "string (required)" }

Response Codes

201 Created - Private offer created successfully
400 Bad Request - Invalid input data
500 Internal Server Error
PATCH /api/privateoffer/{id} Update private offer

Description

Updates an existing private offer with partial data.

Path Parameters

Parameter Type Required Description
id Guid Required Unique identifier of the private offer

Response Codes

200 Success - Private offer updated
400 Bad Request - Invalid input data
404 Private offer not found
500 Internal Server Error
DELETE /api/privateoffer/{id} Delete private offer

Description

Deletes a private offer permanently.

Response Codes

204 No Content - Private offer deleted
400 Bad Request - Cannot delete offer
404 Private offer not found
500 Internal Server Error
GET /api/privateoffer/products-plans Get products and plans

Description

Retrieves all available products with their associated pricing plans.

Response Codes

200 Success - Returns products and plans
500 Internal Server Error

Deal Registration API

DealRegistrationController

8 endpoints

Manage deal registrations and AWS opportunity integrations including creation, updates, stage changes, and invitation handling.

GET /api/dealregistration Get all deal registrations

Description

Retrieves all deal registrations with optional pagination.

Query Parameters

Parameter Type Required Description
page integer Optional Page number for pagination
size integer Optional Number of items per page
top integer Optional Limit number of results
GET /api/dealregistration/{id} Get deal registration by ID

Description

Retrieves a specific deal registration by opportunity ID.

POST /api/dealregistration Create new deal registration

Description

Creates a new deal registration with AWS Partner Central integration.

Query Parameters

Parameter Type Required Description
isDraft boolean Optional Save as draft (default: true)

Request Body

{ "customerCompanyName": "string (required, max 120 chars)", "customerIndustryVertical": "string (required)", "customerWebsite": "string (required, valid URL)", "customerCountry": "string (required)", "customerZipCode": "string (required, max 20 chars)", "projectTitle": "string (required, max 255 chars)", "customerBusinessProblem": "string (required, 20-2000 chars)", "solutionsOffered": "string (required)", "estimatedMRR": "string (required, numeric)", "targetCloseDate": "string (required, YYYY-MM-DD)", "partnerContactFirstName": "string (required, max 80 chars)", "partnerContactLastName": "string (required, max 80 chars)", "partnerContactEmail": "string (required, valid email)" }
PUT /api/dealregistration Update deal registration

Description

Updates an existing deal registration.

Query Parameters

Parameter Type Required Description
isDraft boolean Optional Save as draft (default: true)
POST /api/dealregistration/change-stage Change deal stage

Description

Changes the stage of an existing deal registration.

Request Body

{ "opportunityId": "string (required)", "stage": "string (required)" }
GET /api/dealregistration/invitations Get pending invitations

Description

Retrieves all pending AWS engagement invitations.

POST /api/dealregistration/invitations/accept Accept invitation

Description

Accepts an AWS engagement invitation.

Request Body

{ "invitationId": "string (required)" }
POST /api/dealregistration/invitations/reject Reject invitation

Description

Rejects an AWS engagement invitation with optional reason.

Request Body

{ "invitationId": "string (required)", "rejectionReason": "string (optional)" }
